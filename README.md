## üõ†Ô∏è The Engine: Technology Stack

GalaxyEd is built on a modern, robust, and scalable technology stack:

- **Frontend:** React & TypeScript for a type-safe, component-based architecture.
- **Styling:** TailwindCSS with custom themes for a beautiful, responsive, and consistent user experience.
- **AI Core:** Google Gemini API (`gemini-2.5-pro`, `gemini-2.5-flash`, `veo-3.1-fast-generate-preview`) for all intelligent features.
- **Extensibility:** A custom `multiModelService` designed to integrate various LLMs, including prepared hooks for services like Huawei Cloud.
- **Client-Side Storage:** `localStorage` is used for persisting user profiles, libraries, and tasks, ensuring a seamless experience across sessions.

## üìñ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [repository-url]
    ```
2.  **Install dependencies:** This project uses ES modules imported via CDN in `index.html`, so no `npm install` is required for the main libraries.
3.  **Environment Variables:** Create a `.env` file in the root directory and add your Google AI API key:
    ```
    API_KEY=YOUR_GEMINI_API_KEY
    ```
4.  **Run the application:** Serve the `index.html` file using a local server. A simple way is to use a VS Code extension like "Live Server".
